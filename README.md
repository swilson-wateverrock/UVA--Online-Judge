## Running Locally
JAVA
```sh
$ javac Main.java
$ java Main
```
PYTHON
```sh
$ python Main.py
```
PYTHON with input and output
```sh
$ python Main.py < in.txt > out.txt
```
C++
```sh
$ g++ -o Main Main.cpp
$ ./Main
```
